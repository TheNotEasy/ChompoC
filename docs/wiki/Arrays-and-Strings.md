# Функции и области видимости

## Объявление и вызов

```javascript
fun multiply(left, right) {
    return left * right;
}

var result = multiply(6, 7);
```

Количество аргументов должно точно совпадать с количеством параметров пользовательской функции. Native-функции могут принимать диапазон аргументов.

## First-class functions

```javascript
fun add(a, b) { return a + b; }
var operation = add;
print(operation(2, 3));
```

## Замыкания

```javascript
fun make_counter(start) {
    var value = start;

    fun next() {
        return ++value;
    }

    return next;
}
```

Вложенная функция сохраняет окружение, существовавшее в момент её объявления.

## Области видимости

Каждый `{ ... }`, вызов функции и каждая итерация `for-in` создают окружение. Поиск имени идёт от текущего окружения к родительским и затем к глобальному.

```javascript
var value = 1;
{
    var value = 2;
    print(value); // 2
}
print(value); // 1
```

## Рекурсия

Рекурсия поддерживается, но глубина вызовов ограничена `ChompoConfig::MaxCallDepth` (по умолчанию 512). Превышение даёт контролируемую ошибку `Runtime StackOverflow`.

## Ограничения управляющих операторов

`return` разрешён только внутри функции. `break` и `continue` относятся только к циклам в той же функции: функция, объявленная внутри цикла, не получает право использовать их.
