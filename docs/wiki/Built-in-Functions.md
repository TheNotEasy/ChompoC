# Встроенные функции

Все функции ниже являются callable-значениями: их можно присваивать переменным и передавать в пользовательские функции. Инструкция `print(...)` описана отдельно, потому что она не является callable.

## Преобразования

### `Int(value)`

Возвращает `integer`.

- `bool`, `integer`, `char` преобразуются к целому числу;
- `double` усекается к нулю;
- строка должна целиком содержать корректное целое число;
- `NULL` преобразуется в `0`;
- слишком большое, бесконечное или неподдерживаемое значение вызывает runtime-ошибку.

```javascript
print(Int("42")); // 42
print(Int(3.9));  // 3
print(Int('A'));  // 65
```

### `Double(value)`

Возвращает `double`.

- принимает числовые значения и `char`;
- строка должна целиком содержать корректное число;
- `NULL` преобразуется в `0.0`.

### `Bool(value)`

Возвращает truthiness значения.

Ложные значения: `NULL`, `false`, числовой ноль, пустая строка, пустой массив и `\0`. Остальные значения истинны.

### `String(value)`

Возвращает строковое представление значения. Callable преобразовывать в строку нельзя.

Примеры представления:

```javascript
String(NULL);          // "NULL"
String(Array{1, 2});   // "{1, 2}"
String(true);          // "true"
```

### `Char(value)`

Возвращает байтовый `char`.

- строка должна иметь длину ровно один байт;
- `integer` должен находиться в диапазоне `0..255`;
- `char` возвращается без изменения.

### `Array(value)`

Возвращает массив.

- существующий массив возвращается как тот же ссылочный объект;
- строка преобразуется в массив `char`;
- любое другое значение оборачивается в массив из одного элемента.

```javascript
Array("abc"); // {a, b, c}
Array(42);    // {42}
```

### `CATS(array)`

`Char Array To String`. Преобразует массив, состоящий только из `char`, в строку.

```javascript
CATS(Array{'o', 'k'}); // "ok"
```

### `Type(value)`

Возвращает имя runtime-типа:

```text
NULL
bool
integer
double
char
string
array
callable
```

## Последовательности и массивы

### `len(sequence)`

Принимает `string` или `array` и возвращает число байтов/элементов.

```javascript
len("abc");       // 3
len(Array{1, 2}); // 2
```

### `push(array, values...)`

Мутирует массив, добавляя один или несколько элементов в конец, и возвращает новую длину.

```javascript
var values = Array{};
print(push(values, 10));         // 1
print(push(values, 20, 30));     // 3
print(values);                   // {10, 20, 30}
```

Сигнатура требует минимум два аргумента: массив и хотя бы одно добавляемое значение.

Перед изменением выполняется полная проверка всех новых значений. Если добавление создаст прямой или косвенный цикл массивов, операция завершается ошибкой и исходный массив не меняется.

### `pop(array)`

Удаляет и возвращает последний элемент массива. Для пустого массива возвращает `NULL`.

```javascript
var values = Array{1, 2};
print(pop(values)); // 2
print(values);      // {1}
print(pop(values)); // 1
print(pop(values)); // NULL
```

Из-за ссылочной семантики изменения через `push/pop` видны всем aliases массива.

## Ввод и файлы

Полное описание находится на странице [Ввод и вывод](Input-and-Output).

| Функция | Аргументы | Назначение |
|---|---:|---|
| `input()` | 0 | прочитать строку, `NULL` на EOF |
| `istream(path?)` | 0..1 | сменить входной поток |
| `ostream(path?, mode?)` | 0..2 | сменить поток `print` |
| `iostream(inputPath?, outputPath?, mode?)` | 0..3 | сменить оба потока |

## TCP Network API

Полное описание находится на странице [Network API](Network-API).

| Функция | Аргументы | Результат |
|---|---:|---|
| `netListen(host, port, backlog?)` | 2..3 | listener handle |
| `netConnect(host, port)` | 2 | socket handle |
| `netAccept(listener)` | 1 | socket handle или `NULL` |
| `netPoll(handles, timeoutMs?)` | 1..2 | массив готовых handles |
| `netSend(socket, data)` | 2 | число отправленных байт |
| `netReceive(socket, maxBytes?)` | 1..2 | пакет `data/wait/closed` |
| `netReceiveLine(socket)` | 1 | строковый пакет `data/wait/closed` |
| `netPort(handle)` | 1 | локальный порт |
| `netClose(handle)` | 1 | `NULL` после закрытия |

## `print(arguments...)`

`print` — инструкция языка. Она вычисляет аргументы слева направо и выводит их подряд без автоматических разделителей.

```javascript
print("x=", 10, "\n");
```
