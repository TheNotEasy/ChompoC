# Встроенные функции

Все встроенные функции являются обычными callable-значениями и могут сохраняться в переменные.

## Преобразования

| Функция | Аргументы | Результат |
|---|---:|---|
| `Int(value)` | 1 | `integer`; `NULL → 0`, `bool/char → integer`, `double` усекается, строка разбирается целиком |
| `Double(value)` | 1 | `double`; `NULL → 0.0`, числа и `char` преобразуются напрямую, строка разбирается целиком |
| `Bool(value)` | 1 | `bool` по правилам truthiness |
| `String(value)` | 1 | строковое представление; callable преобразовывать нельзя |
| `Char(value)` | 1 | `char` из
