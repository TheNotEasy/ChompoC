# Типы и операторы

## Типы

| Имя `Type(value)` | Описание |
|---|---|
| `NULL` | отсутствие значения |
| `bool` | `true`/`false` |
| `integer` | знаковое 64-битное целое |
| `double` | число с плавающей точкой |
| `char` | один байт `0..255` |
| `string` | последовательность байтов |
| `array` | изменяемый массив значений со ссылочной семантикой |
| `callable` | пользовательская или native-функция |

## Truthiness

Ложными считаются `NULL`, `false`, числовой ноль, пустая строка, пустой массив и `Char(0)`. Остальные значения истинны.

## Арифметика

`+`, `-`, `*`, `/`, `%`. `bool` и `char` участвуют в числовых операциях как целые. Если один операнд `double`, результат числовой операции обычно `double`.

Дополнительно:

```javascript
"ab" * 3;                 // "ababab"
Array{1, 2} * 2;           // {1, 2, 1, 2}
"value=" + 10;            // "value=10"
Array{1} + Array{2};       // {1, 2}
```

## Сравнения

`==`, `!=`, `<`, `<=`, `>`, `>=`. Совместимые числовые типы сравниваются по числовому значению. Массивы сравниваются глубоко.

## Логика

`!`, `&&`, `||`. `&&` и `||` используют short-circuit и возвращают `bool`.

## Принадлежность

```javascript
2 in Array{1, 2, 3};
'a' in "chat";
"hat" in "chat";
```

Для массива применяется та же семантика, что у `==`. Для строки левый операнд должен быть `char` или `string`.

## Присваивание и обновление

`=`, `+=`, `-=`, `*=`, `/=`, prefix/postfix `++` и `--`. Целью может быть переменная или индексируемый элемент:

```javascript
values[0] += 10;
text[1]++;
```

## Приоритет от низкого к высокому

присваивание → `||` → `&&` → `== !=` → сравнения и `in` → `+ -` → `* / %` → unary → вызов/индексация/postfix.
